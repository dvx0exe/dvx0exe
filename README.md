### 🚀 Projetos em Destaque

#### 🤖 [Java Groq Integration](https://github.com/dvx0exe/java-groq-integration)
Aplicação em **Java 11+** que integra a API da **Groq (Llama 3)** para geração de bibliografias estruturadas. 
- **Destaque:** Uso de *JSON Mode* para saída de dados estruturados e desserialização automática com Jackson.
- **Tecnologias:** Java, Maven, Jackson, Groq API.

#### ⚔️ [Elden RPG Character Manager](https://github.com/dvx0exe/Elden)
Um sistema interativo de gerenciamento de personagens inspirado no universo de *Elden Ring*.
- **Funcionalidades:** Lógica de herança de classes, persistência de builds e interface intuitiva.
- **Acesse online:** [dvx0exe.github.io/Elden/](https://dvx0exe.github.io/Elden/)
- **Tecnologias:** JavaScript, HTML5, CSS3.

#### 🐍 [Beginning of Elden (CLI)](https://github.com/dvx0exe/beginning-of-elden)
Sistema robusto de gerenciamento de RPG via terminal focado em lógica de Back-end.
- **Destaque:** Implementação de segurança (hashing de senhas), manipulação de inventário em JSON e persistência em banco de dados relacional.
- **Tecnologias:** Python, MySQL, Dotenv.
